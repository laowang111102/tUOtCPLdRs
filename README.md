# 前言

大家好，本次分享的毕业设计项目是基于Java语言和MySQL数据库开发的攀枝花市鲜花销售系统。此项目具备实战意义，涉及前端、后端及数据库的全面开发。以下是项目详细介绍，希望能对您的学习和实践有所帮助。

## 内容介绍

本项目旨在为攀枝花市的鲜花销售行业提供一个便捷、高效、实用的在线销售平台。系统主要包括用户注册登录、鲜花浏览、购物车、订单管理、后台管理等模块。通过此项目，用户可以在线选购心仪的鲜花，实现快速下单；商家可以及时处理订单，提高工作效率。

## 技术介绍

- **语言：** Java
- **使用框架：** Spring Boot
- **前端技术：** JS、Vue、CSS3
- **开发工具：** IDEA/Eclipse
- **数据库：** MySQL 5.7/8.0
- **数据库管理工具：** phpstudy/Navicat
- **JDK版本：** jdk1.8
- **Maven：** apache-maven 3.8.1-bin
- **前端环境：** Node.Js 12\14\16
